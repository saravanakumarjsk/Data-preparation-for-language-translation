# Data-preparation-for-language-translation
To prepare data for language translation.

## Description
Language transalation is one of the core component of modern day machine learning. This algo uses few techniques to clean and prepare data before feeding them into the network.

## About dataset
This is a sample data set splitted by me, the original one is over 300 mb.
It has conversations carried out in the parliment in german and english.

## Execution 
To run this file clone this repo to your local machine and do
> `pyhton german_to_english.py`
